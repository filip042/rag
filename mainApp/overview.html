RAG application providing document indexing and LLM-based question answering.

The project is organized into the following modules:

  • rag-core — shared abstractions for LLM providers, vector stores, and document parsers
  • rag-app — main application handling indexing, retrieval, and the web frontend
  • rag-starter-provider-ollama — Ollama LLM provider integration
  • rag-starter-provider-openai — OpenAI LLM provider integration
  • rag-starter-llm-deepseek-ollama — DeepSeek-R1-specific LLM provider integration
  • rag-starter-llm-qwen3-ollama — Qwen3-specific LLM provider integration
  • rag-starter-llm-gpt4o-openai — GPT-4o-specific LLM provider integration
  • rag-starter-vectorstore-elasticsearch — Elasticsearch vector database provider integration
  • rag-starter-parser-tika — Apache Tika document parser provider integration
  • rag-starter-parser-markdown — Markdown document parser provider integration
  • rag-integration-tests — OpenAI LLM provider integration
